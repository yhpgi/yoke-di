AndroidSampleWorker

class AndroidSampleWorker(appContext: Context, params: WorkerParameters) : CoroutineWorker

A sample Android Worker that demonstrates non-composable dependency injection. Yoke is initialized at app startup via the YokeContentProvider, making global injection available throughout the app lifecycle.
